# DJProxy adblock — curated domain-only blocklist
#
# A CONSERVATIVE subset distilled from the StevenBlack unified-hosts project
# (https://github.com/StevenBlack/hosts — MIT), reduced to bare ad/tracker
# DOMAINS only (no `0.0.0.0` prefix, no IPs, no localhost redirects). Kept small
# on purpose: this rides the hot CONNECT path, and an over-broad list would
# false-positive-block first-party CDNs. Parent-suffix matching in Blocklist.kt
# means one entry (e.g. `doubleclick.net`) also covers every sub-domain
# (`ad.doubleclick.net`, `stats.g.doubleclick.net`, …), so we list registrable
# domains, not every host.
#
# Format: one domain per line, lowercase. `#` starts a comment (whole-line or
# trailing). `0.0.0.0 domain` / `127.0.0.1 domain` hosts-file lines are also
# accepted and normalized. Blank lines ignored.
